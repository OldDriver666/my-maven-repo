# maven-repo
My personal maven repository.

## Usage
pom.xml:
```xml
    <repositories>
        <repository>
            <id>davis-maven-repo</id>
            <url>https://gitee.com/zhangruhong/maven-repo/raw/master/repository</url>
        </repository>
    </repositories>
```
 **注意 这个地址是下载二进制文件的地址 不是浏览器访问路径 区别在raw**
